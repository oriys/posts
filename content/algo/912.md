## 解法一：快速排序递归

```python
class Solution:
    def sortArray(self, nums: List[int]) -> List[int]:
        if nums is None or len(nums) == 0:
            return []
        return self.sortArray(list(filter(lambda x: x <= nums[0], nums[1:]))) + [nums[0]] + self.sortArray(list(filter(lambda x: x > nums[0], nums[1:])))
```
